My recent Projects


Cocktail Generator

This app lets users input ingredients into the search engine, and an AI API generates unique recipes based on the submitted ingredients. Users can enter any ingredient, and the API collects data to generate a customised response. Developed with HTML, CSS, and JavaScript, this project is a cocktail generator offering a seamless recipe creation experience.


Weather App

This app allows users to explore weather details for cities worldwide. Using a weather API, it provides real-time information like current temperatures, humidity, wind conditions, and weekly forecasts. Developed with React.js and integrated Weather API, it offers an immersive exploration of weather data


World Clock App

This application enables users to discover and compare timezones among a curated list of cities. By integrating a timezone API, it dynamically showcases various times and dates for the selected cities, alongside the user's current location, ensuring automatic updates.
